## ğŸš€ How to Use

### View Tool (see folder tree)
1. Open `View/index.html` in your browser
2. Click "Select Folder" and pick any folder
3. See your folder tree instantly
4. Click buttons to save as text or picture

### Create Tool (make folders from text)
1. Open `Create/index.html` in your browser
2. Paste a tree structure (like the one above)
3. Click "Select Target Folder" (where to create)
4. Click "Generate" - folders and files appear!
